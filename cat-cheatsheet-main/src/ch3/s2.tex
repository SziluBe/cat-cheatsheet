\wde{3.2.X Cardinality} Given sets $A,B$, we write $|A| = |B|$, and say that $A$ and $B$ \textbf{have the same cardinality}, if $A \cong B$.
\wde{3.2.X $\le$ for Sets} Given sets $A$ and $B$, write $|A| \le |B|$ if there exists an injection $A \to B$. We write $|A| < |B|$ if $|A| \le |B|$ but $|A| \ne |B|$.
\wt{3.2.1 Cantor-Bernstein} Let $A$ and $B$ be sets. If $|A| \le |B| \le |A|$ then $A \cong B$ (and vice-versa).
\wt{3.2.2 Cantor} Let $A$ be a set. Then $|A| < |\cat{P}(A)|$.
\wc{3.2.3} For every set $A$, there is a set $B$ such that $|A| < |B|$.
\wpr{3.2.4} Let $I$ be a set, and let $(A)_{i \in I}$ be a family of sets. Then there exists a set not isomorphic to any of the sets $A_i$.
\wde{3.2.X Small/Large Class} A \textbf{class} is informally a collection of mathematical objects. A class is \textbf{small} if it is a set, and \textbf{large} otherwise.
\wde{3.2.X Small/Large Category} A category $\cat A$ is \textbf{small} if the class/collection of all maps in $\cat A$ is small, and \textbf{large} otherwise. A category is \textbf{locally small} if for each $A,B \in \cat A$, the class $\cat A(A,B)$ is small. A category is \textbf{essentially small} if it is equivalent to some small category. 
\wde{3.2.X Homset} The class $\cat{A}(A,B)$ is the \textbf{homset}.
\wpr{3.2.8} $\Set$ is not essentially small.
\wde{3.2.10 Cat} We denote by $\Cat$ the category of \emph{small} categories and functors between them. The set of \emph{all} categories and functors is $\CAT$.
